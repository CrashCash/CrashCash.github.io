Tracer 9 GT+ software licenses

This was obtained by going to Applications ⇒ Settings ⇒ System ⇒ Legal ⇒ 3rd party licenses in the bike's user interface, making a video and clicking down on the joystick over 600 times (once per line) then using mpv and lios to do OCR to convert it to text, and EMACS to convert it to HTML.
This is formatted exactly as displayed on the bike, including typos and word-order errors, except I have linkified the URLs for your viewing pleasure. Note that a lot of these links don't actually work.

Notable items:
  • Python & perl - scripting languages. These are not actually installed, and the Python license is the old incorrect BeOpen license, not the current Python Software Foundation one.
  • connman - COVESA Diagnostic Log and Trace framework
  • Qt5 - user interface development framework
  • OpenSSL - standard cryptographic library
  • readline, liblinenoise - not sure why these are useful on a device without a keyboard
  • MiracleCast - probably how the Garmin navigation software displays on the bike
  • Mosquitto - MQTT lightweight IoT messaging protocol broker
  • Ubiquitous QuickBoot - Linux/Android fast boot solution
  • Gracenote media ID/metadata management - present, but no copyright notice
  • FrogLogic Squish commercial cross-platform GUI and regression testing tool - present, but no copyright notice
  • There's no copyright notice for the Linux kernel, systemd, wayland, or other userland
  • It's a shock to see Slashdot Media mentioned on a consumer motorcycle. I grew up with Slashdot being the coolest and nerdiest of all the tech websites.
